### New composable: `useBackButton`

`src/composables/useBackButton.js` — generic composable for Android/browser back button overlay dismissal:
- Accepts `isOpen` (Ref\<boolean\>), `close` (function), and `{ mobileOnly?: boolean }` options
- Pushes `history.pushState` sentinel when overlay opens
- On `popstate`: closes overlay without calling `history.back()` again
- On programmatic close: calls `history.back()` to pop the sentinel
- `mobileOnly`: only pushes sentinel when `window.matchMedia('(max-width: 768px)').matches`
